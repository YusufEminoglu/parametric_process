Quick start guide

How to use Parametric Process Studio

1

Prepare a QGIS layer

Select a polygon vector layer that represents parcels or blocks. A projected CRS in meters gives accurate metric areas.

2

Launch the cockpit

Start the browser cockpit from the QGIS plugin dock. Viewport features load in real-time.

3

Select a parcel

Click any parcel in 3D to tune setback, height, usage, and typologies in the inspector.

4

Run Evolutionary Optimization

Switch to Evolutionary Studio tab, choose NSGA-II/SPEA-2, select objectives, and run multi-objective optimization.
